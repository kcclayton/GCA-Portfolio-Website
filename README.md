# Project: Portfolio Website
### The following includes assignment details as provided by GCA instructors.
Time to bring your vision to life on screen!

Leverage AI and your knowledge of HTML and CSS to build a simple, single-page personal website to showcase your projects. You can update this page with every project you create, in this course and beyond, creating an impressive portfolio to showcase your abilities. 

Use Copilot or ChatGPT to help you brainstorm design ideas, apply web development best practices, and troubleshoot issues. 

Side note: don't worry if at this point you feel like AI is doing most of the work. Your input and understanding of the fundamentals will become more important as you progress through the projects to more complex work. For now, the goal is to have fun and get a 'quick win', inspiring and empowering you for all the great projects ahead! 

**Important: while similar to the bio card you created alongside Guil as you followed along with previous videos, this project is more complex. Do not submit just a bio card - follow the instructions to create a full web page please!**
